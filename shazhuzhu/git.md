## 快速入门(20180410)
- 安装
    - https://git-scm.com/
- 设置
```
$ git config --global user.name "Your Name"
$ git config --global user.email "email@example.com"
```
- clone
    - `git clone https://git.coding.net/coolfe/carom-in-2018.git`
- 内容
    - `git status`
    - `git add`
    - `git commit -m 'some text'` 
    - `git pull`
    - `git push`
    - `git log`
    - `git checkout`
