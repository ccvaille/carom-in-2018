# ls:list
- ls: 打印出当前目录的清单
- ls -a: 列出所有文件，包括隐藏文件和隐藏文件夹
- ls -l: 使用长格式列出文件
- ls -t: 按照文件建立时间列出文件 
- ls -alt
# pwd
- pwd: 查看当前工作目录
# cd
- cd dirName: 切换到 dirName目录
- cd .. : 返回上一层
# touch
- touch name: 新建名为 name 的文件
# mkdir: mark directory
- mkdir dirName: 新建 dirName 目录
# cp:
- cp 
# mv:移动文件或者将文件重命名
- mv fileName fileName: 重命名文件
- mv fileName ... dirName: 移动文件
